## UX principles (Hush Gentle)

### Brand tone
- Calm, reassuring, informative — **never pushy**.
- Trust-first copy: transparency > hype.
- No medical claims.

### Visual system
- Pastel neutrals, high whitespace, clean grid.
- Rounded corners, minimal shadows, subtle transitions.
- Large, readable typography with clear hierarchy.

### Mobile-first behavior
- Tap targets ≥ 44px.
- Clear sticky actions (e.g., Add to Cart on PDP is a large primary CTA).
- Lightweight menu (simple mobile nav).

### Accessibility basics
- Visible focus ring (`:focus-visible`).
- Semantic headings and labels.
- Calm error messages (short, human).

### Practical patterns borrowed from references (not copying branding)
- **82°E / Kimirica**: clean hero, strong whitespace, minimal card grid, gentle tone.
- Product pages: large imagery, short benefit line, clear CTA, trust indicators.
